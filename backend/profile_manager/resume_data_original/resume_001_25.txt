Docker 기반의 컨테이너화된 환경에서 개발 및 배포를 진행하였고 시스템을 관리하였습니다.

문제 해결 능력 : 모니터링 에이전트가 늘면서 성능에 대한 이슈가 생겼습니다. 이것을 해결하기 위해서 리팩토링 및 버전업 작업을 진행했습니다. 우선, 대시보드에서 중복 데이터 및 API를 주기적으로 호출하고 있었습니다. 대시보드 설정값은 localStorage에 넣어두고 중복 데이터를 없애기 위해 서버의 모델 및 API 모두 수정하였습니다. 그 결과 5초마다 12개의 API를 2MB씩 받아오던 것을 6개의 API 및 800KB로 줄일 수 있었습니다. scatter 차트에서는 기존 최대 1만 개에서 3만 개로 늘어나도 문제없이 작동하게 되었습니다.
커뮤니케이션 : 팀 내 팀장님의 퇴사로 패스워드 관리프로그램 업무를 맡았던 경험이 있습니다. 에이전트 작업 및 기능개발, 현대오일 프로젝트 수행을 위해서 이사님, 이전 팀장님, 그리고 담당자와 원활한 소통을 통해 프로젝트를 성공적으로 완료하고 추가적인 설치 작업까지 진행하였습니다.
기술 스택 다양성: 주기적으로 세미나를 하면서 AI, Typescript, React, Elasticsearch, GraphDB, Spring batch 등 기술 스택의 다양성을 쌓았습니다. 다양한 기술 스택을 이해하고 있으며, 필요에 따라 빠르게 새로운 기술을 배우고 적용하는 데 열린 마음을 가지고 있습니다.
프로젝트 관리: 첫 프로젝트인 국가기록원 블록체인 사업이 4개 컨소시엄으로 구성된 프로젝트였습니다. 우리 회사가 마더업체로서 프로젝트 일정을 관리했습니다. 대전, 부산 등을 다니면서 일정 및 RFP 요구사항을 맞추기 위해 노력하였고 완료보고까지 큰 이슈 없이 마무리하였습니다.
창의성과 혁신 : 회사에서 자바스크립트만 사용하고 있을 때 앞으로 유지보수 및 프로젝트 관리를 위해서 React, Vue를 비교해보고 Vue 도입을 하였습니다. 회사 내부 Vue 교육을 하고 여러 팀의 요구사항을 반영한 웹서비스를 위한 Vue 프로젝트를 만들었습니다.
